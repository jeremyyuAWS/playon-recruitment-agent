# Compliance Artifacts Directory

This directory stores generated compliance reports and audit artifacts.

## Structure

```
compliance/
├── eeo/                    # EEO-1 annual reports
│   └── eeo1-2026.json      # Generated by compliance-reporting skill
├── ofccp/                  # OFCCP compliance artifacts
│   ├── applicant-flow/     # Applicant flow logs per requisition
│   ├── adverse-impact/     # Adverse impact analysis reports
│   └── good-faith-efforts/ # Documentation of outreach efforts
├── pay-equity/             # Pay equity audit results
│   └── audit-2026-q1.json  # Generated by compensation-benchmarking skill
├── i9-audit/               # I-9 verification tracking
│   └── reverification-log.json
└── audit-trail/            # System action logs
    └── agent-actions.jsonl  # Every automated decision logged
```

## Retention Policy
- EEO-1 reports: 3 years minimum
- Applicant flow logs: 2 years (OFCCP requirement)
- Pay equity audits: 3 years
- I-9 records: 3 years from hire OR 1 year from termination
- Agent action logs: 2 years

## Access Control
- **HR Director**: Full access
- **Legal**: Full access
- **Recruiters**: Applicant flow logs only
- **Managers**: No direct access (request via HR)
- **Agents**: Write-only (generate reports, cannot modify past records)

## Audit Trail Format
Each agent action is logged as a JSON line:
```json
{
  "timestamp": "2026-03-11T14:30:00Z",
  "agent": "talent-acquisition",
  "action": "stack_rank_candidates",
  "job_id": "LEV-12345",
  "candidates_processed": 2500,
  "candidates_advanced": 200,
  "decision_rationale": "Scored against JD requirements using standard rubric",
  "human_review_required": false
}
```

All automated decisions that affect candidate outcomes MUST be logged.
