# Onboarding Workflow

## Instructions

Manage the complete onboarding journey from offer acceptance through the first 90 days.

### Pre-Start (Offer Accepted → Day 1)
- [ ] Send welcome packet (handbook, benefits enrollment forms, I-9)
- [ ] IT ticket: laptop, monitors, peripherals ordered
- [ ] Access provisioning: email, Slack, GitHub, Lever, relevant tools
- [ ] Assign onboarding buddy (same team, 6+ months tenure)
- [ ] Schedule Day 1 orientation
- [ ] Notify hiring manager with start date and onboarding plan

### Week 1
- [ ] Day 1 orientation: company overview, values, org chart
- [ ] IT setup verification: all tools accessible
- [ ] Meet the team: 1:1s with direct teammates
- [ ] Role-specific training sessions scheduled
- [ ] Buddy intro lunch/coffee

### Day 30 Check-In
- [ ] Manager 1:1: alignment on expectations and initial projects
- [ ] Buddy check-in: comfort level, questions, blockers
- [ ] Self-assessment: confidence rating (1-5) on key responsibilities
- [ ] Flag any access or tooling gaps

### Day 60 Check-In
- [ ] Project contribution review with manager
- [ ] Expanded 1:1s with cross-functional partners
- [ ] Training completion status
- [ ] Buddy check-in: integration into team culture

### Day 90 Review
- [ ] Formal 90-day review with manager
- [ ] Buddy program graduation
- [ ] Employee satisfaction survey
- [ ] Flag any retention risks
- [ ] Transition to regular performance cycle

### Automation
- Send automated reminders for each checkpoint to manager + buddy + new hire
- If any checklist item is overdue by 3 days, escalate to HR
- Track completion rates for onboarding health metrics
