# Candidate Sourcing

## Instructions

Proactively identify and engage passive candidates for hard-to-fill roles.

### When to Activate
- Role open for 14+ days with fewer than 50 qualified inbound applications
- Hiring manager requests proactive sourcing
- Niche/specialized role with small talent pool

### Sourcing Channels
1. **Internal talent pool** — Previous applicants who scored 60+ but weren't selected
2. **Employee referrals** — Generate referral requests to relevant teams
3. **LinkedIn Recruiter** — Build boolean search strings from job requirements
4. **Job board databases** — Indeed, Dice, BuiltIn (for tech roles)
5. **University pipelines** — For entry-level and intern roles

### Process
1. Analyze job description for must-have vs. nice-to-have skills
2. Build search queries optimized for each channel
3. Generate candidate shortlist with preliminary match scores
4. Draft personalized outreach messages (not generic templates)
5. Track response rates and optimize messaging

### Outreach Template Structure
```
Subject: [Specific role detail] at PlayOn Sports
- Why them specifically (reference their work/skills)
- What makes the role interesting (not generic company pitch)
- Clear next step (15-min intro call)
- Easy opt-out
```

### Metrics
- Response rate target: >25% for personalized outreach
- Qualified response rate: >15%
- Track source-of-hire to optimize channel spend
